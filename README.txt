alternatif lain.
B. MEMBUKA DENGAN MENGAKTIFKAN WEB SERVER
   1. aktifkan webserver (xampp, lampp, atau sejenisnya)
   2. pindahkan semua file dan folder (dalam satu folder nb) ke folder public server (htdocs)
   3. buka browser masukan alamat (localhost/nb/index.html)
   4. secara otomatis akan memberikan notifikasi untuk donwload file.
   5. pilih simpan untuk mendownload.
